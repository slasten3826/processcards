# Legacy

Статус:

```text
historical documentation branch
not current canon
```

Сюда перенесены документы предыдущих поколений `ProcessCards`.

Они сохраняются:

- как история проекта
- как semantic archive
- как след design-эволюции

Но их нельзя читать как current canon
для новой machine-ветки.

## Structure

- [table/](./table/) — old gameplay/table branches
- [crystall/](./crystall/) — old implementation slices and handoff docs
- [manifest/](./manifest/) — old identity, visual, and prototype notes

## Reading law

If a legacy doc conflicts with current canon in:

- `docs/table/`
- `docs/crystall/`
- `docs/manifest/`

then current canon wins.
