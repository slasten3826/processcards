# Reset Law

Статус:

```text
canonical table law
adopted individual trump event
```

Legacy breadcrumb:

```text
XVI Tower
```

Alternate machine-facing title:

```text
HARD RESET
```

Core edge:

```text
☲ -> ☶
CYCLE -> LOGIC
```

## 1. Core identity

`RESET` is a catastrophic hand-reset trump.

It does not improve the current hand gently.
It destroys the current hand as a built structure
and replaces it with a new one.

Short formula:

```text
hand-state collapse -> forced replacement
```

## 2. Reading

`☲` here is not soft cycle.

It is full turnover:

- release current hand
- take new hand

`☶` here makes that turnover non-optional,
hard,
and structurally violent.

So this trump reads as:

```text
cycle under rupture
```

Not:

```text
filter
```

And not:

```text
gentle rebuild
```

## 3. Resolution

When `RESET` becomes known:

1. discard all cards from hand to grave
2. perform `6` draw procedures
3. if any draw procedure reveals a trump, that draw procedure burns into ordinary trump event flow
4. resolve exposed trumps in arrival order according to the normal trump flow order
5. continue unfinished draw of `RESET` after each trump in flow resolves, unless another resolved effect explicitly prevents continuation
6. after `RESET` fully resolves, `RESET` itself follows ordinary trump ecology

Short formula:

```text
discard hand
draw 6
drawn trumps resolve by normal trump flow
RESET follows normal trump ecology
```

## 4. Why draw 6

The effect is intentionally:

```text
draw 6
```

Not:

```text
draw the same number
```

Reasons:

- deterministic
- does not depend on current hand size
- does not scale with prior hand inflation
- reads as replacement, not recycling

`RESET` should feel like a violent state rewrite,
not card-neutral filtering.

## 5. Relation to draw law

`RESET` does not bypass normal draw semantics.

It uses:

- ordinary draw procedures
- ordinary trump burn on trump reveal
- ordinary continuation of remaining draw procedures

См.:

- [DRAW_PROCEDURE_LAW.md](./DRAW_PROCEDURE_LAW.md)

## 6. Gameplay role

Early game:

- harsh disruption
- forced rebuild
- hand-state break

Mid game:

- tempo swing
- rebuild point
- risk of losing a carefully shaped hand

Late game:

- endgame destabilizer
- trump-cascade trigger through draw
- collapse-driven climax

Short formula:

```text
RESET pushes the game from stable machine-building into collapse-driven turbulence
```

## 7. Relation to general trump law

`RESET` does not invent its own ecology.

It uses:

- ordinary trump flow
- ordinary ordered resolution
- ordinary chain close handling
- ordinary trump-zone law

This matters.

`RESET` should stress the machine,
not escape it.

Short formula:

```text
RESET does not break the machine
RESET proves the machine is there
```

## 8. Design character

This trump should feel:

- severe
- mandatory
- non-negotiable
- dangerous
- cleansing in a violent way

It is not reward.
It is not smoothing.
It is not value padding.

It is:

```text
catastrophic replacement of hand-state
```

## 9. Minimal canonical text

```text
Discard your hand. Perform 6 draw procedures.
If a draw procedure reveals a trump, it enters ordinary trump event flow.
After RESET fully resolves, it follows ordinary trump ecology.
```
