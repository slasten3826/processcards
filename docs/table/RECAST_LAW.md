## 1. Core identity

`RECAST` is a manifest-chain inversion trump.

It does not create value directly.
It rewrites the relation between:

- current visible world
- current hidden underside
- current hand as intervention surface

Short formula:

```text
old visible world becomes hand
hidden world becomes visible
old hand becomes new hidden underside
overflow becomes residue
```

## 3. Temporary term

This event uses a temporary holding zone:

```text
proto-hand
```

`proto-hand` is not a persistent game zone.

It is only a temporary event container
for the old visible manifest row
while the rest of the recast is happening.

## 4. Core law

When `RECAST` becomes known:

1. move all six cards from the current face-up `manifest` row into `proto-hand`
2. reveal all six cards in the current `latent` row
3. those six revealed latent cards become the new face-up `manifest` row
4. shuffle the player's current hand
5. deal up to six cards from that shuffled hand into `latent[1..6]`
6. if fewer than six cards were available from hand, fill the remaining latent slots from deck with preservation of current information state
7. reveal all remaining cards from the shuffled hand one by one in reveal order
8. route each revealed overflow card by class:
   - non-trump -> grave
   - trump -> ordinary trump flow
9. move all cards in `proto-hand` into the player's hand
10. after `RECAST` fully resolves, `RECAST` itself follows ordinary trump ecology

Short formula:

```text
manifest -> proto-hand
latent -> manifest revealed
old hand -> latent
old hand overflow -> class routing
proto-hand -> hand
RECAST -> ordinary trump ecology
```

## 5. Resulting state

After resolution:

- `manifest` = former latent row
- `latent` = up to six cards from old hand, with deck completion if needed
- `hand` = former visible manifest row
- old hand overflow = written into residue by reveal order
- `targets` = unchanged
- `runtime` = unchanged

Short zone formula:

```text
manifest becomes hand
latent becomes manifest
hand becomes latent
overflow becomes residue
```

## 6. Overflow truth

This part is load-bearing.

Overflow from the old hand is not silently discarded.
It is revealed one by one
and then routed by class.

That means:

- non-trump overflow writes ordered grave
- trump overflow enters ordinary trump flow

This keeps `RECAST`
inside the same machine truth
as every other reveal-triggered trump path.

`RECAST` does not invent separate ecology.

- structural
- world-inverting
- non-local

because it changes what counts as:

- world
- hidden support
- intervention surface
