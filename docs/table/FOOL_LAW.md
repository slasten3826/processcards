# Fool Law

Статус:

```text
canonical table law
adopted individual trump event
```

Legacy breadcrumb:

```text
0 Fool
```

Core edge:

```text
▽ -> ☰
FLOW -> CONNECT
```

## 1. Core identity

`FOOL` is a deck-drilling trump.

It does not search with precision.
It falls forward through deck
until trump contact is made.

Short formula:

```text
drill through deck until trump contact
```

## 2. Reading

`▽ FLOW` here is continuous forward movement.

`☰ CONNECT` is not assembly of a chosen relation.
It is contact with the next trump-event hidden in deck.

So this trump reads as:

```text
movement continues until event-connection is found
```

This is not:

```text
search
```

And not:

```text
selection
```

It is:

```text
advance until connection
```

## 3. Resolution

When `FOOL` becomes known:

1. reveal the top card of deck
2. if that card is not a trump, move it to top of grave
3. repeat from the new top card of deck
4. when a revealed top card is a trump, that trump enters ordinary trump event flow
5. after `FOOL` fully resolves, `FOOL` itself follows ordinary trump ecology

Short formula:

```text
reveal topdeck
non-trump -> grave
repeat
trump -> ordinary trump event flow
FOOL -> ordinary trump ecology
```

## 4. Why this works

`FOOL` does not create value by selecting the best card.

It creates movement pressure.

It cuts through ordinary deck surface
and keeps going until trump-layer contact happens.

This makes it:

- simple
- dangerous
- difficult to fully predict
- naturally stronger in trump-rich late states

## 5. Relation to trump ecology

`FOOL` does not invent a separate ecology.

It uses:

- ordinary topdeck reveal
- ordinary ordered resolution
- ordinary trump flow
- ordinary chain close handling
- ordinary trump-zone law after its own resolution

Its specificity is not in ecology.
Its specificity is in how it reaches the next event.

## 6. Design character

`FOOL` should feel:

- forward
- reckless
- simple
- archetypal
- less controlled than `EJECT`
- less structural than `RECAST`
- more like entry into motion than command of the machine

It starts the road.
It does not yet justify it.

## 7. Minimal canonical text

```text
Reveal the top card of deck.
If it is not a trump, move it to grave and repeat.
If it is a trump, send it into ordinary trump event flow.
After FOOL fully resolves, it follows ordinary trump ecology.
```
