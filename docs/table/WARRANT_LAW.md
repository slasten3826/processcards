# Warrant Law

Статус:

```text
canonical table law
adopted individual trump event
```

Legacy breadcrumb:

```text
IV Emperor
```

Core edge:

```text
☰ -> ☴
CONNECT -> OBSERVE
```

## 1. Core identity

`WARRANT` is a lawful acquisition trump.

It does not draw blindly.
It does not merely observe.
It does not tutor through impossible deck-depth.

It grants direct sanctioned access.

Short formula:

```text
take any card on board into hand
```

## 2. Reading

`☰ CONNECT` here is not ordinary hand refill.

It is access by connection.
Authority reaches the card, not chance.

`☴ OBSERVE` here is not private inspection only.

It means the card is not taken by chaos,
but by sanctioned access to what is already present in machine reality.

So this trump reads as:

```text
authorized access becomes acquisition
```

This is not:

```text
search
```

And not:

```text
random draw
```

It is:

```text
sanctioned seizure of the already-existing
```

## 3. Core law

When `WARRANT` becomes known:

1. take one card on board into hand
2. if the taken card is a trump, it enters ordinary trump event flow instead of remaining in hand
3. `WARRANT` itself then follows ordinary trump ecology

Short formula:

```text
take one board card
if trump -> ordinary trump event flow
WARRANT -> ordinary trump ecology
```

## 4. Scope reading

`Any card on board` must be read through `BOARD_SCOPE_LAW`.

This means:

- `targets`
- `manifest`
- `latent`
- `runtime`
- `grave`
- `trump flow`
- `trump zone`
- `topdeck`

But not:

- `hand`
- deep `deck` beyond topdeck

Board zones still remain governed by their own visibility and reach laws.

So this card is broad,
but still bounded by the actual structure of the game.

Short formula:

```text
WARRANT grants board access
not impossible access
```

## 5. Information consequence

If `WARRANT` reaches a hidden card,
it does not bypass the information model silently.

The card must become known as part of being taken.

If that card is a trump,
ordinary trump-event rules then take over.

This keeps `WARRANT` inside:

- [BOARD_SCOPE_LAW.md](./BOARD_SCOPE_LAW.md)
- [CARD_INFORMATION_STATE_LAW.md](./CARD_INFORMATION_STATE_LAW.md)
- [TRUMP_EVENT_MINIMAL_LAW.md](./TRUMP_EVENT_MINIMAL_LAW.md)

## 6. Why this works

`WARRANT` is strong because it bypasses ordinary card flow.

It does not wait for draw.
It does not only inspect.
It converts permission directly into acquisition.

This makes it:

- blunt
- powerful
- highly flexible
- dangerous when it pulls trumps
- very different from ordinary minor economy
- not replacement for `EJECT`, but a different reach vector

## 7. Relation to trump ecology

`WARRANT` does not invent a separate ecology.

It uses:

- ordinary known-state trigger
- ordinary trump flow if a trump is pulled
- ordinary ordered resolution
- ordinary chain close handling
- ordinary trump-zone law after its own resolution

Its specificity is not in ecology.
Its specificity is in how reach is granted
without requiring destruction.

## 8. Restrictions

`WARRANT` does not:

- secretly ignore physical zone structure
- bypass impossible deck-depth access
- create extra hidden state
- suppress immediate trump resolution when a trump is taken
- destroy the zone it reaches into

It grants access.
It does not grant annihilation.

## 9. Design character

`WARRANT` should feel:

- lawful
- mechanical
- sanctioned
- blunt
- less mystical than `FOOL`
- less destructive than `EJECT`
- less explosive than `RESET`
- more like a machine-issued right of seizure
- more like access than intervention

This is intentional.

## 10. Minimal canonical text

```text
Take one card on board into hand.
If it is a trump, it enters ordinary trump event flow instead of remaining in hand.
After WARRANT fully resolves, it follows ordinary trump ecology.
```
