## 1. Form

Равносторонний треугольник, вершина вверх.

```text
△
```

- Только контур (stroke), без заливки.
- Без внутренних деталей.
- Толщина линии: пропорционально размеру глифа.

## 2. Ink

Тёмный ink (graphite / charcoal), выбирается по контрасту с полем карты.

MANIFEST-поле — золотое (`{0.86, 0.68, 0.26}`), светлое → тёмный глиф читается уверенно.

Ink не зашит в геометрию глифа. Цвет передаётся параметром при отрисовке.

Minor и trump карты используют один и тот же ink для этого глифа.
